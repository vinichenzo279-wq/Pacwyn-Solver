# Pacwyn Solver

Pacwyn Solver is a browser-based toolkit for building, analysing, and playing PACWYN squads. It combines draft solving, RTG and Weekend League team building, SBC help, live-match guidance, mini-games, formation analysis, and editable player and manager databases in one standalone app.

## Features

### Draft Solver
- Choose a formation and enter the five options shown for each draft slot.
- Get recommendations for which card to take or hold.
- Build the strongest final XI from the cards you selected.
- Add optional squad conditions when the result must meet specific requirements.
- Import supported draft screenshots for a faster, reviewable setup.

### Conditions
Create squad requirements such as:
- Minimum or exact team rating and chemistry.
- Cards from selected nations, leagues, or teams.
- Limits for Icons, Heroes, special cards, ratings, and card types.
- Required players or exact cards.

Condition lists can be imported and exported for reuse.

### RTG / Weekend League
- Build an optimised XI from the player database.
- Compare formations and manager options.
- Use presets for different squad-building priorities.
- Filter available cards and exclude cards or managers you cannot use.
- Build specifically to face a known opponent team.

### Draft Cups
- Optimise a draft squad before choosing a manager.
- Compare the five manager offers and rebuild around the best available option.
- Keep the actual cards selected from each draft pick.

### Live Play
- Import or enter your squad and track a match as it happens.
- Record revealed opponent cards, stats, chemistry clues, and manager boosts.
- Receive score-aware recommendations for the next duel.
- Use exact endgame analysis when only a few cards remain.
- Export match and squad information for later use or bug reports.

### SBC Solver
- Find an XI that meets all active conditions.
- Prefer lower-rated solutions after a valid squad has been found.
- Require or exclude exact cards and player identities.
- Redo a result without losing your current condition setup.

### Mini-Games
Helpers are included for:
- **Hidden Player**
- **Kick Word**
- **Player Links**

Supported screenshots can be imported and reviewed before clues are applied.

### Formation, Weights, and Pivot Analysis
- Compare available formations.
- Rank useful nations and leagues to build a draft around.
- Adjust helper priorities or use database-based calibration.
- Browse and compare individual cards.

### Editable Databases
- Search, add, edit, remove, import, and export player cards.
- Maintain manager chemistry and stat boosts.
- Keep the app usable as PACWYN adds new cards and collections.

## Getting Started

1. Download `index.html`.
2. Open it in a modern browser.
3. Choose the section you want from the navigation bar.
4. Enter your cards, conditions, or squad information.
5. Run the relevant solver or analysis.

There is no installation or build process. The app can also be hosted directly with GitHub Pages.

Most features run locally in the browser. Screenshot-reading tools load external OCR resources and therefore require an internet connection when used.

## Mobile Use

The interface is designed for both desktop and mobile browsers. Large searches may take longer on mobile devices, especially on iOS. Use the stop or cancel controls when available rather than closing the page during a search.

## Data and Saves

The player and manager databases are included inside the HTML file. Settings, boards, and completed results may be stored in the browser for convenience. Updating to a new app version can reset saved local state.

## Reporting Problems

Please report:
- Incorrect or missing cards.
- A valid squad that the solver failed to find.
- An invalid squad returned as a solution.
- Incorrect Live Play or Mini-Game results.
- Broken controls, screenshot imports, or mobile behaviour.

Include the app version, the inputs used, screenshots where useful, and an exported solver or recreation log when available.
